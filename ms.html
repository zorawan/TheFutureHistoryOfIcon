History

The floppy disk

Imagine embarking on a journey to explore the evolution of user interface design over time. A journey that takes you back to the good old days when the floppy disk icon was the universal symbol for saving files. This iconic icon was an essential element in early user interface design and was universally recognized. Designers had to work with it to deliver an explicit affordance for the "save" button. However, the floppy disk icon has disappeared from most user interfaces with the advent of newer and more advanced ways of saving files. But the journey doesn't end here. We explore the impact of this transition on user experience and how designers have adapted to the changing times. Together, we'll discover how icons shape the user experience over time and hopefully predict these selected icons' future.

Takeaways

Universal icons which do not pass the user testing

  • Folder

    Folder is not clear with its meaning because people confused between it and FILE.

  • Rate

    Star has many meanings according to the context, it mainly used as FAVORITE, LOVE, SAVE. It is not always RATE.

  • Message

    Text bubble can be one way action as comment and two way communication like message and chat. It is hard to determine the clear meaning and the action.

  • Edit

    Pencil has many meanings according to the context, it represents any actions using pen or pencil. Ex. EDIT, DRAW, WRITE, POST, SIGN. It also can be an object(s) like NOTES, POSTS.

Skeuomorphic icons that fail to accurately represent their intended meaning.

  • Piggy Bank

    The piggy bank, traditionally a symbol of saving money, has become less clear in its representation over time. As people no longer use it as frequently to save money, it has taken on various interpretations in the realm of finance and lost some of its symbolic association with the action of saving.

  • hourglass

    The hourglass used to be the timer, but it now has many meanings in a digital interface, such as loading, waiting, and processing. Moreover, almost all these meaning has been replaced by a progress bar and loading animation like a spinning wheel.

  • SD Card

    The SD card, which used to be a common memory storage device similar to flash drives and floppy disks, has become less necessary with the widespread adoption of cloud storage. Additionally, some people may be confused by its shape, which resembles a SIM card.

"The assumption that a universal icon will always be universally understood may not hold true, as the presence of obsolete objects can impact people's comprehension of digital interfaces."
